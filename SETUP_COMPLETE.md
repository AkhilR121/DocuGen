## Quick Start Guide

### Prerequisites

**Client:**
- Node.js 18+ ✅ (Need to upgrade from v12)
- npm 9+

**Server:**
- Python 3.11+ ✅ (You have Python 3.14)
- Poetry ✅ (Installed - v2.4.1)

### Running the Server

#### Option 1: Using Startup Scripts (Recommended)

**Windows:**
```bash
cd server
start_server.bat local
```

**Mac/Linux/Git Bash:**
```bash
cd server
./start_server.sh local
```

#### Option 2: Using Poetry Directly

```bash
cd server
poetry run python run.py --env local
```

#### Option 3: Different Environments

```bash
# Local development
poetry run python run.py --env local

# Development server
poetry run python run.py --env dev

# Production
poetry run python run.py --env prod
```

### Running the Client

```bash
cd client
npm install  # Install dependencies
npm run dev  # Start dev server
```
